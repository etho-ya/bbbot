## 📊 Пример при следующей сделке

**Ваши настройки:**
- Баланс: ~53 USDT
- Депозит в сделку: 10%
- Плечо: 30x

**Сигнал:** WIF LONG, вход 0.3385

**Что сделает бот:**
```
Маржа = 53 × 10% = 5.3 USDT
Размер позиции = 5.3 × 30 = 159 USDT  ✅
Qty = 159 / 0.3385 = ~470 монет
```

**В Bybit увидите:**
- IM (маржа): ~5 USDT
- Value (позиция): ~159 USDT ✅
- Qty: ~470 WIF
